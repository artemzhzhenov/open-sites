# open-sites

Открытые страницы. Живут на [artemzhzhenov.xyz](https://artemzhzhenov.xyz).

## НЕ ЗЕМЛЯ

Десять мест, снятых на этой планете, которые выглядят как чужие миры. Скролл
ведёт камеру внутрь кадра: место наплывает, держится ровно столько, чтобы
прочитать подпись, и читатель проходит сквозь него — а из центра уже растёт
следующее. Внизу всё это время идёт счётчик километров, и в конце он замирает
на 82 786 км: два экватора, не покидая Земли.

Статическая страница без сборки — `index.html`, `styles.css`, `main.js`,
`assets/`. Ни одной зависимости, ни одного сетевого запроса в рантайме:
открывается двойным кликом по файлу. Без JS остаются десять обычных
полноэкранных секций; при `prefers-reduced-motion` нырок не включается вовсе.

Снимки — Wikimedia Commons под свободными лицензиями. Авторы и лицензии указаны
в финале страницы и в [`assets/CREDITS.md`](assets/CREDITS.md). Расстояния между
точками — по большому кругу, R = 6371 км, и накопленное ровно равно сумме
показанных этапов.

Шрифты: Unbounded, Golos Text, JetBrains Mono (SIL Open Font License).
